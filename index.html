Features

続けたくなる、仕組みがある

習慣化を科学した機能が、あなたのモチベーションを維持します。

⚔️

LvUp RPGシステム

習慣をこなすたびにXP獲得。Lv1「見習い冒険者」からLv50「神話の英雄」まで、ゲーム感覚で成長を実感。

📊

ヒートマップ統計

GitHub風のヒートマップで継続状況を一目で確認。週・月・年単位のデータ分析で自分の傾向を把握できる。

🎯

AIゴールマッチング

「月100km走る」などの目標を設定すると、AIが記録を自動マッチングして進捗を更新。目標達成をサポート。

💬

LINEで記録

LINEで「💪胸 17:00-18:00」と送るだけで記録完了。Googleカレンダーへの追加リンクも自動生成される。

🌐

タイムライン＆SNS

フォロワーの記録がタイムラインに表示。いいね・コメント・メンションで互いに刺激し合いながら継続できる。

📅

カレンダー連携

GoogleカレンダーのiCalを連携してTodoを自動取り込み。チェックすると記録まで一気に完結。

📤

SNSシェア

Todo完了時にX・LINE・Threads・Instagramへワンタップシェア。達成感を仲間と共有しよう。

🔒

プライバシー設定

非公開モードをオンにすると、フォロワー以外には記録データが見えなくなる。安心して記録を続けられる。

📱

スマホ完全対応

PCでもスマホでも快適に使えるレスポンシブデザイン。通勤中・就寝前のさっと記録に最適。

How to Start

3ステップで始められる

難しい設定は一切なし。今すぐ使い始められます。

1

アカウント登録

ユーザー名とパスワードだけで30秒で完了。メール認証不要。

2

カテゴリを選んで記録

「📝 記録」から今日の活動を入力。細かい設定は後からでOK。

3

成長を確認・継続

ヒートマップとLvUpでモチベーションを維持しながら継続しよう。
